# Document AI Vector Search System

This project allows users to upload any document, extract its content,
store embeddings in a vector database (Chroma), and query the document
using AI.

## Features
- Upload any document
- Extract text
- Store embeddings in vector DB
- Ask questions from documents
- Web UI
- Ask any query from any document which are upload we can upload any king of document like research, technincal, or any.

## Tech Stack
- Python
- Flask
- LangChain
- LangGraph
- ChromaDB
- HTML

## Run Project

pip install -r requirements.txt 
open cmd at root - folder
python app.py
